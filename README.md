# FutebolPT
[This project is not actively maintained anymore, since widgets are no longer supported by macOS.]

## About
*FutebolPT* is a macOS widget that provides information about the Portuguese main football championship.
It list the matches and results of all rounds of the championship, as well the current standings.

It uses as source the website of the [Liga Portuguesa de Futebol Profissional (Portuguese League for Professional Football)](https://www.ligaportugal.pt/).
